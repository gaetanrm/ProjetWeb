Comment naviguer sur MinuteMeal ?close
    - Vous pouvez rechercher grâce à la barre de recheche située en haut à gauche
    - Si vous souhaitez voir ou ajouter une recette, il suffit d'aller sur l'onglet Recettes
    - Si vous souhaitez voir tous les ingrédients, vous devez cliquer sur l'onglet Ingrédients
    - Enfin, si vous souhaitez vous connecter ou vous inscrire, cliquez sur Connexion
